
# ContinuumCare Game (MVP Skeleton)

## Quick start
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

This is a minimal React + Vite + TypeScript skeleton with:
- Mode Select
- Case View with Patient Panel & Therapy Cards
- Simple rules engine demo (eGFR gate, peri-op SGLT2 hold)
- Debrief banner and scoring stub

Data files live in `public/data`.
